[To do App Screenshot](https://www.dropbox.com/s/muiqhvv2hokz6zj/todo-app.png)

## Build Setup

```bash
# install dependencies
$ npm install

# run gulp from local npm
$ npm run gulp

# if you have globaly installed gulp
$ gulp watch
```
